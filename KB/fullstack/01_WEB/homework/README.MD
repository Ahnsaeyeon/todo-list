# - 플로우 차트

```mermaid
graph TD;
    A[페이지 로드] --> B[저장된 데이터 로드];
    B --> C[현재 날짜 출력];
    C --> D[사용자 입력 대기];

    D -->|할 일 추가| E[새로운 아이템 생성 및 저장];
    E --> F{마감일 확인 (오늘인지?)};

    F -- 아니요 --> G[완료/삭제 대기];
    F -- 예 --> H[마감일 강조 표시] --> G;

    G -->|완료 버튼 클릭| I[완료 상태 토글];
    G -->|삭제 버튼 클릭| J[아이템 삭제];

    I --> K[로컬 스토리지 데이터 저장];
    J --> K;
```
